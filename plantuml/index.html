    Using PlantUML to create UML diagrams from text

    Example

    Setup tools

    Run local plantuml server

    docker run -d --name plantuml -p 8081:8080 plantuml/plantuml-server:tomcat

    To test out, go to http://localhost:8081

    Obsidian

    • Install plugin plantuml
    • Set server url to http://localhost:8081

    Visual Studio Code

    • Install PlantUML VS Code plugin
    • In jebbs.plantuml plugin setting set plantuml.server to http://localhost:8080
    • Allow http on localhost

    Mkdocs

    mkdir
mkdocs new .

    site_name: My Docs
plugins:
    - plantuml:
        puml_url: http://localhost:8080

    rm -r mkdocs-venv; virtualenv mkdocs-venv; . mkdocs-venv/bin/activate; pip install mkdocs==1.5.3 mkdocs-puml ; mkdocs serve
