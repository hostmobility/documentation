    WiFi (MX-4)

    Overview

    T30 FR

    • This product is using Texas Instruments WL1837MOD (wlink8) WL1837MOD.
    • Interface name is wlan0
    • Standard: IEEE 802.11b/g/n

    T30 and CT

    • This product is using Zcomax Technologies ZCN-722M (ath9k).
    • Interface name is wlan0
    • Standard: IEEE 802.11b/g/n

    For Linux 3.1 system

    Note: for bsp release 1.6(1.5) and older

    Usage

    All that is required to start Wifi moudule is ifup wlan0.

    #/etc/networking/interfaces
auto wlan0
iface wlan0 inet dhcp
    wpa-driver nl80211
    wpa-conf /etc/wpa_supplicant.conf

    One has to edit /etc/wpa_supplicant.conf to add new networks which the module should connect to. 

    #/etc/wpa_supplicant.conf
ctrl_interface=/var/run/wpa_supplicant
ctrl_interface_group=0
update_config=1

network={
        ssid="HostMobility-Guest"
        psk="*************"
        scan_ssid=1
}

    See more configuration examples at http://linux.die.net/man/5/wpa_supplicant.conf

    scan wifi networks

    iw dev $WIFI_INTERFACE scan

    Wifi as Access Point

    Target setup

    Run 

    opkg update
opkg install hostap-daemon
opkg install iperf
opkg install dhcp-client

    Example script

    This example include a way to test diffrent channels and diffrent WLAN modes and messure it with iperf. conect to the SSID test and setup a iperf server with iperf -s on the connected PC.

SEND_TIME_IN_SEC=$1

UDHCPD_CONFIG_FILE=/tmp/udhcpd.conf
HOSTAPD_PID=0
# Add or remove channel or hw_mode
CHANNEL=("1" "7" "11")
HW_MODE=("b" "g")

if [[ -z $SEND_TIME_IN_SEC ]]; then
    SEND_TIME_IN_SEC=120
fi

ip link set wlan0 down
pkill udhcpd
pkill hostapd
############################################
if [[ -f "/etc/init.d/hostapd" ]]; then
    echo "/etc/init.d/hostapd file exist continue test"
else
      echo "Install hostap-daemon again"
      exit 1

ip addr add 192.168.20.100/24 dev wlan0
ip route add default via 192.168.20.1

cat > $UDHCPD_CONFIG_FILE <<'EOF'

# The start and end of the IP lease block
start       192.168.20.20
end     192.168.20.254  
# The interface that udhcpd will use
opt    router  192.168.20.100
interface   wlan0

pidfile /var/run/udhcpd-wlan0.pid   

EOF

udhcpd -S ${UDHCPD_CONFIG_FILE}
if [ $? != "0" ]; then
    echo "Install udhcpd server again(dhcp-client)"
    exit 1

else
    echo "udhcpd server initialized"
fi
